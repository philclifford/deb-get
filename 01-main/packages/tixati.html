Download Tixati v3.14 for Linux

The Linux version of Tixati has been well-tested on Fedora, Ubuntu, Mint, and Mandriva Linux.  It should also work on almost any other recent Linux distro as long as GTK2 is installed.

Both 32-bit and 64-bit versions of Tixati are available.  To run the 64-bit version, you must have both a 64-bit processor and a 64-bit operating system.  If you are unsure, the 32-bit version will run well in either environment as long as the 32-bit GTK2 libraries are present.

To report bugs or make comments, please visit the Feedback and Support forum.
